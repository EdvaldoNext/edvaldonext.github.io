                    BRINCANDO DE DOS

                    TIME - ALTERA A HORA
                    DATE - ALTERA DATA
                    VER - VERSÃO DO SISTEMA
                    MEM - MEMÓRIA
                    VOL - VOLUME DO SISTEMA OU SEJA O NOME DAS UNIDADES EX UNIDADE C
                    label - altera o nome do volume
                    CLS - limpa TELA
                    dir - visualiza o que tem nas pasta, e também cria arquivos dentro do diretório ex: dir>nome_do_arquivo_ea extensao do arquivo ex: txt, docx,etc
                    : - troca de unidade c: ou f: ou G:
                    md - cria pasta
                    CD - ENTRA NA PASTA (quando vc tem uma pasta com espaço em branco vc precisa dar " eo nome da pasta)
                    CD .. - VOLTA PÁGINA
                    CD\ - VOLTA PASTA RAIZ
                    TREE - COMANDO MOSTRA A ÁRVORE DA PASTA
                    ENTRA DENTRO DE PASTA ESPECIFICA CD \ MAIS PASTA PAI\ MAIS PASTA FILHO
                    EDIT - EDITOR DE TEXTO ex EDIT ESPAÇO NOME DO ARQUIVO OBS: USAR ALT PARA NAO USAR MOUSE
                    REN - COMANDO QUE RENOMEIA O ARQUIVO OU SEJA DA A EXTENSÃO EX ren nome do arquivo espaço nome do novo arquivo
                    copy - copia arquivo ex: copy nome do arquivo espaço caminho do novo arquivo e pasta que deseja copiar
                    copy todos os arquivos - vc entra na pasta e da o seguinte comando ex: copy *.nome da pasta, espaço,caminho e pasta
                    * - significa tudo
                    copy arquivos de extensões ex copy *.doc ,espaço, caminho e pasta
                    move - mesmo coisa do copy so que move ao invés de copiar
                    del - apaga arquivos ex: del espaço nome do arquivo
                    (del/q/f/a/s * .  ”tipo do arquivo”)
                    rd - apaga pasta vazias
                    rd /? - pede ajuda ao rd, rd /s remove os arquivos
                    rd /q remove os arquivos sem pedir permissão
                    ctrl e c - interrompe o que o dos esta executando
                    xcopy copia pastas inteiras
                    help - relações de comando pausado ex help | more
                    attrib - exibe todos os arquivos e seus atributos ou seja ate os ocultos, voce tambem pode deixar o arquivo oculto
                    atraves dos atributos ex: atrib +h nome do arquivo,(- tira, + coloca) ver atributos /?
                    type - visualiza o arquivo de texto sem ter que abrir
                    PING = ping e como se fosse um medidor de velocidade de rede
              (abra cmd digite ping mais o endereço ) ele forneço informação
                    como quantos dados foram enviados e recebidos e uma estatística
                    do tempo médio em comparação com mínimo e o alto, quanto menor o
                    tempo melhor.
                    Tracert = traça o caminho do computador ate o servidor
                    Tasklist = exibe todos os arquivos em execução
                    Obs: e possível parar programas em execução com o comando pause e o nome do programa

                    Dir = mostrar pasta ocultas do sistema = dir /a:hs

                    Systeminfo = carrega informações do sistema
                    wmic baseboard get product,manufacturer = placa mae e fabricante para baixar drives
                    robocopy: backup de arquivos ex = robocopy (caminho da pasta a ser copiada) espaço (caminho do destino da copia)

                    reinicializar com modo de segurança Windows 8
